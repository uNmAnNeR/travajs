## Features
* Node and Browser support
* easy to use
* no external dependencies
* customizable and extendable

## Install
`npm install trava` and `import Trava from 'trava';`

or use CDN:

`<script src="https://unpkg.com/trava"></script>`

## Build & Test
`npm run make`

## Compatibility
Supports all major browsers and IE11+
